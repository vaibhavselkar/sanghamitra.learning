Understanding Processes

What is a Process?

A process is a series of steps taken to achieve a specific goal. We follow processes daily:

  • Making tea
  • Commuting to college
  • Sending a message

Why Break Down Processes?

  • Reduces complexity
  • Minimizes errors
  • Enables progress tracking

Real-World Example: Making Tea

  1. Gather ingredients (tea, water, sugar, milk)
  2. Boil water in kettle
  3. Add tea leaves to boiling water
  4. Add sugar and stir
  5. Pour milk and simmer
  6. Strain into cup

Problem Set: Calculator Program

Objective: Create basic arithmetic calculator

Requirements

  • Input: Two numbers + operator
  • Output: Operation result
  • Handle division by zero
  • Validate operator input

Implementation Steps

  1. Get user inputs (num1, operator, num2)
  2. Validate operator
  3. Perform calculation
  4. Handle special cases
  5. Display result
if operator == '+':
    result = num1 + num2
elif operator == '-':
    result = num1 - num2
elif operator == '*':
    result = num1 * num2
elif operator == '/':
    if num2 != 0:
        result = num1 / num2
    else:
        print("Error: Division by zero")
