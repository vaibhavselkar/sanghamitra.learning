Computational Thinking Principle

Loop Advantages

  • Automate repetitive tasks
  • Reduce redundant code
  • Enhance efficiency and scalability

Real-Life Examples

  • Daily attendance marking
  • Mass production of bricks

Loop Implementation Process

Flowchart Representation

  1. Start → Initialize condition
  2. Check condition → Continue or Exit
  3. Execute task → Update condition
  4. Repeat from step 2

Pseudocode Example

       Set count to 1
        While count ≤ 5:
            Print "Hello World!"
            count += 1

Best Practices

Good Naming

attempts = 0
        while attempts < 3:
            print("Processing...")
            attempts += 1

Bad Naming

x = 0
        while x < 3:
            print("Looping")
            x += 1

Common Errors

  • Infinite loops from missing updates
  • Off-by-one errors in range()
  • Modifying lists during iteration
