ℹ️ Composite Function Answer Formatting Guide

How to format your answers:

  • Composite Functions: f(g(x)), (f∘g)(x), f(g(2)), f∘g(x)
  • Function Notation: f(x) = 2x + 1, g(x) = x^2, f∘g(x) = 2x^2 + 1
  • Domains/Ranges: domain: x ≥ 0, range: (-∞, ∞), all real numbers
  • Composition Results: (f∘g)(x) = x^2 + 3, f(g(3)) = 10, g(f(x)) = (x+1)^2
  • Inverse Functions: f^-1(x), f⁻¹(x) = (x-1)/2
  • Math Symbols: sqrt(x) = √x, ^ = exponent, ∘ = composition, pi = π, inf = ∞
  • Intervals: (-∞, 3) or (-inf, 3), [0, ∞) or [0, inf)

Composite Function Examples:

(f∘g)(x) = (x^2 + 1)^2
→ (f∘g)(x) = (x² + 1)²
f(g(2)) = 5
→ f(g(2)) = 5
f⁻¹(x) = (x-3)/2
→ f⁻¹(x) = (x-3)/2
domain: x ≠ 0, x ≥ 1
→ domain: x ≠ 0, x ≥ 1

Important:

  • Type sqrt(2) to get √2
  • Type x^2 for x²
  • Type f^-1 or f⁻¹ for inverse
  • Type inf for ∞
  • Use ∘ or type "o" for composition
