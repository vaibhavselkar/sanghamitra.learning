ℹ️ Sets & Relations Answer Guide

How to format your answers:

  • Set Notation: {1,2,3} = $\{1,2,3\}$, ∅ = $\emptyset$, ℕ, ℤ, ℝ = $\mathbb{N}, \mathbb{Z}, \mathbb{R}$
  • Set Operations: A ∪ B = $A \cup B$, A ∩ B = $A \cap B$, A ⊆ B = $A \subseteq B$
  • Relations: {(a,b), (c,d)} = $\{(a,b), (c,d)\}$, xRy = $x R y$, R = {(x,y) | x+y=5} = $R = \{(x,y) \mid x+y=5\}$
  • Math Symbols: ∈ = $\in$, ∉ = $\notin$, ∀ = $\forall$, ∃ = $\exists$
  • Logic Symbols: ∧ = $\land$, ∨ = $\lor$, ¬ = $\lnot$, ⇒ = $\Rightarrow$

Sets and Relations Examples:

{x ∈ ℕ | x ≤ 5}
→ $\{x \in \mathbb{N} \mid x \leq 5\}$
{(1,2), (2,3), (3,4)} → $\{(1,2), (2,3), (3,4)\}$
A ∩ B = {x | x ∈ A ∧ x ∈ B}
→ $A \cap B = \{x \mid x \in A \land x \in B\}$
∀x ∈ A, ∃y ∈ B: xRy
→ $\forall x \in A, \exists y \in B: x R y$

Important:

  • For sets: Use {1,2,3} or {x | x > 0}
  • For relations: Use {(a,b), (c,d)} or {(x,y) | x^2 + y^2 = 1}
  • For set operations: Use A ∪ B, A ∩ B, A - B
  • Use Unicode or LaTeX for special symbols
