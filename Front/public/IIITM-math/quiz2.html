ℹ️ Answer Formatting Guide

How to format your answers:

  • Infinity: Type inf for ∞ or -inf for -∞
  • Square roots: Type sqrt(x) for √x
  • Equations: Type x = 2, x = 3 for multiple solutions
  • Decimals: Round to 3 decimal places if needed
  • Fractions: Use a/b or (a)/(b)
  • Exponents: Use ^ like x^2 for x²

Examples:

Type: (-inf, 3]
Shows: $(-\infty, 3]$
Type: x = -1/2, x = sqrt(2)
Shows: $x = -\frac{1}{2}, x = \sqrt{2}$
Type: {x | x ≠ 0}
Shows: $\{x \mid x \neq 0\}$

Important Notes:

  • Answers are case-insensitive
  • Spaces are automatically removed
  • See preview below input box for how your answer will be interpreted
  • Multiple answers should be separated by commas
