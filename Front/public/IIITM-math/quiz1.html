ℹ️ Answer Formatting Guide

How to format your answers:

  • Math Symbols: inf = ∞, sqrt(x) = √x, ^ = exponent, ∪ = union, ∩ = intersection
  • Set Notation: {1,2,3}, {x | x > 0}, ∅ = empty set
  • Intervals: (-∞, 3] = $(-\infty, 3]$
  • Equations: x = -1/2, sqrt(2) (multiple answers with commas)
  • Functions: f(x) = x^2 + 1, y = mx + b
  • Polynomials: x^2 - 4x + 4, (x-2)^2

Quick Examples:

(-inf, 3] → $(-\infty, 3]$
x = -1/2, sqrt(2) → $x = -\frac{1}{2}, \sqrt{2}$
f(x) = x^2 + 2x + 1 → $f(x) = x^2 + 2x + 1$
A ∪ B = {1,2,3} → $A ∪ B = \{1,2,3\}$

Important:

  • Case-insensitive, spaces auto-removed
  • Multiple answers separated by commas
  • Check preview below input box
  • Use calculator button for complex calculations
